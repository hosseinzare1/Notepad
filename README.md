# Notepad

Android notepad app implemented with kotlin

## Tools and features

- Kotlin
- MVP architecture
- Reactive programming (RXjava)
- Dependency injection (Hilt)
- Database (Room)

- Set note priority & category
- Search notes
- Filter notes
